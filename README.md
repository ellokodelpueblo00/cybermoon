# CYBERMOON

Proyecto desarrollado para el evento Avatar Jam "Bolivia Ucrónica 3D".

## Descripción

Este proyecto consiste en crear una reinterpretación libre y ucrónica de un personaje o hito histórico de Bolivia, usando modelado 3D, diseño visual y una landing page de presentación.

## Ramas del proyecto

- main: versión principal del proyecto.
- modelo-3d: avances del modelo 3D.
- landing-page: desarrollo de la página web de presentación.
- bocetos: ideas, referencias y bocetos iniciales.
- documentacion: explicación del proyecto y narrativa.
